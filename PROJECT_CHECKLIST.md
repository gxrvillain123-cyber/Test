## Create GitHub repository

```bash
cd svu-barrackpore-website
git init
git add .
git commit -m "Build SVU Barrackpore landing page"
git branch -M main
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```

If you put this folder inside an existing repository, upload only the website project contents to your new project repo.

## Deploy on Vercel

1. Go to https://vercel.com/new
2. Import your GitHub repository
3. Keep framework as `Next.js`
4. Build command: `npm run build`
5. Output settings: leave default
6. Deploy and copy the hosted URL

## Walkthrough video checklist

Recommended length: 2-5 minutes.

1. Open the hosted link on desktop.
2. Show navbar, hero, CTA buttons, and animated hero card.
3. Scroll through About, Programs, Campus, Testimonials, CTA, and Footer.
4. Click the program tabs: Technology, Health Sciences, Management.
5. Hover over buttons/cards/gallery images to show transitions.
6. Open responsive/mobile view in browser dev tools.
7. Show the mobile menu and scroll the full landing page.
8. End by showing that the page is hosted and responsive.
