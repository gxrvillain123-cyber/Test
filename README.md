# Responsive Developer Portfolio

A production-ready portfolio website built from the PRD with Next.js, Tailwind CSS, and Lucide icons.

## Highlights

- Responsive hero, about, skills, projects, contact, and build decisions sections
- Sticky navigation with active section state and mobile menu
- Project cards with local thumbnails, tech stacks, GitHub links, and live demo links
- Contact form validation that opens an email draft after valid input
- SEO metadata with Open Graph and Twitter preview data
- Local resume PDF and visual assets in `public/portfolio-assets`

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Customize Content

Update the profile, skills, projects, links, and build notes in `lib/portfolio-data.ts`.
